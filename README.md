📝 Smart To-Do List App

A simple and interactive To-Do List web app built using Streamlit.

🚀 Features
- Add, Delete, Update Tasks
- Mark tasks as completed
- Set deadlines 📅
- Assign priority ⚡
- Track progress 📊

🛠 Tech Stack
- Python
- Streamlit
